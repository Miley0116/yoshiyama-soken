# 吉山創建株式会社 ウェブサイト

神奈川県の外構工事、土木工事、リフォームを手がける吉山創建株式会社のコーポレートサイトです。

## 🔧 最新修正内容（2024.01.08 v1.9）

### 原因究明と修正
1. **ハンバーガーメニューのバツボタン**
   - **原因**: translate値が5pxで大きすぎた
   - **修正**: translate値を4pxに変更
     - translate(5px, 5px) → translate(4px, 4px)
     - translate(5px, -5px) → translate(4px, -4px)
   - **結果**: Xマークが適切なサイズで画面内に収まる

2. **LINE QRコード**
   - **原因**: padding: 8pxでQRコードが小さく見えた
   - **修正**: paddingを6pxに変更（8px → 6px）
     - 白枠サイズは250pxのまま維持
   - **結果**: QRコードが白枠の中央に大きく表示される

## 🚀 使用方法

### 1. ローカルで開く
```bash
# index.htmlをダブルクリック
# または
open index.html  # Mac
start index.html # Windows
```

### 2. 動作確認のポイント
- **デスクトップ表示（968px以上）**
  - ナビゲーションが横並びで表示
  - ヘッダー右側に営業時間が表示
  - ハンバーガーメニューが非表示
  - LINE QRコードがきれいに表示

- **モバイル表示（967px以下）**
  - ハンバーガーメニュー（三本線）が表示
  - クリックでナビゲーションが開閉
  - ヘッダー右側が非表示
  - LINE QRコードがきれいに表示

## 🎨 カスタマイズ

### 会社情報の変更
`index.html` の以下の箇所を編集してください：
- 電話番号：`XXX-XXXX-XXXX` を実際の番号に変更
- 住所：`〒XXX-XXXX 神奈川県横浜市` を実際の住所に変更
- 代表者名：`吉山 太郎` を実際の名前に変更
- Googleマップ：実際の住所のマップURLに変更

### 色の変更
`styles.css` で以下の色を変更できます：
- プライマリカラー: `#3498db`
- テキストカラー: `#2c3e50`
- 背景色: `#f8f9fa`

### 画像の変更
- `images/logo.jpg`: ロゴ画像
- `images/line-qr.jpg`: LINE QRコード
- Unsplash画像URLを変更して背景やサービス画像を変更可能
